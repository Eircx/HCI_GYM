# HCI_GYM
开发者：熊天翼、孟本源、陈小雨、周朔亦

一个在Unity中开发的智能健身房场景，用户可以通过语音、手势和头动的方式和屋内的设备进行交互。项目可在Oculus Quest2设备上运行。

屋内设有跑步机、杠铃、哑铃三个可以交互的运动器材，温湿度控制系统，饮料自动分配系统和音乐系统，同时用户可以通过墙上的数据板观看今日的运动效果。
